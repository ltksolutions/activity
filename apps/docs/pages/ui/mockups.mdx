# UI Mockups

Tento dokument popisuje **kľúčové UI obrazovky** systému. Mockupy sú v textovej / štruktúrovanej forme — finálne pixel-perfect dizajny budú v Figme po zložení tímom dizajnéra. Účelom tu je zachytiť **funkčnú štruktúru**, **navigačné toky** a **prepojenie s doménovým modelom**.

## Globálna navigácia

### Web app (activity.sportup.sk)

Hlavná navigácia v hornej lište:

```
┌────────────────────────────────────────────────────────────────────┐
│  [LOGO sportup]    Domov  Aktivity  Mentoring  Chat       👤▾ 🔔   │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│                          MAIN CONTENT                               │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

- **Logo** — kliknutím Domov
- **Domov** — dashboard pre danú rolu
- **Aktivity** — timeline všetkých aktivít užívateľa
- **Mentoring** — len pre osoby s mentor / mentee rolou; cykly a sedenia
- **Chat** — Courier inbox
- **👤▾** — profil dropdown (môj profil, nastavenia, organizácie, log out)
- **🔔** — notifikácie dropdown s posledných 10 notifikácií + linkom na full inbox

Sub-navigácia per sekcia (kontextová) je v ľavom side panele alebo cez tabs.

### Admin app (admin.activity.sportup.sk)

Iné navigačné štruktúre — orientované na správu organizácie:

```
┌────────────────────────────────────────────────────────────────────┐
│  [LOGO admin] {Org Name ▾}                                  👤▾ 🔔  │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  Dashboard | Členovia | Tímy | Aktivity | Komunikácia | Reporty    │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

### Mobile app

Bottom navigation:

```
[🏠 Domov]  [📋 Aktivity]  [💬 Chat]  [👤 Profil]
```

Pre operácie ako *Vytvoriť sedenie* — floating action button v rohu.

### Cyklus — pohľad mentee

Rovnaké ako mentor, ale:

- Akčná lišta: `[+ Navrhnúť sedenie]  [💬 Otvoriť Courier]  [⋯ Viac]`
- Drafty mentora **nie sú** viditeľné
- Súkromná poznámka mentora **nie je** viditeľná v zozname ani v detaile

## Ďalšie obrazovky
