## Logo

Logo **activity** je postavené na koncepte **Chat Stack** — tri riadky abstraktného zoznamu konverzácií, ktoré zároveň vyjadrujú tri typy komunikácie:

| Riadok | Typ | Farba |
|--------|-----|-------|
| 1 | 1:1 priama správa | SK Blue |
| 2 | Skupinový kanál | SK Blue + SK Red |
| 3 | Broadcast | Ink (utlmený) |

Wordmark: **Albert Sans 700 · lowercase · −0.03em**

## Export PDF

1. Otvoriť `activity-brand-manual.html` v prehliadači
2. Kliknúť **Exportovať PDF** (tlačidlo vpravo dole)
3. `Cmd+P` / `Ctrl+P` → Paper: **A4** · Margins: **None** · Background graphics: **✓**
4. Uložiť ako PDF
